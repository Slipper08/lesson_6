Блочная модель. "Всё - прямоугольник"

Каждый элемент на странице - это прямоугольник. Во время отрисовки браузеры применяют к ним CSS-свойства.

Что же такой блочная модель? По сути это то, как браузер считает размеры элемента. За метод расчета отвечает параметр "box-sizing". Возможные значения следующие:

  • content-box - значение по умолчанию. Размеры объекта вычисляются следующим образом:
    • ширина: значение width контентной части + padding*2 + border*2
    • высота: значение height контентной части + padding*2 + border*2
  • border-box - можно назначить принудительно. В этом случае значения width и height будут включать в себя padding и border.

Демонстрация работы box-model

Что получается:

В случае значения box-sizing: content-box ширина блока считается как ширина контентной части (100) + оступы слева и справа (10+10) и граница слева и справа (5 и 5) = 130 px. Для высоты тоже самое.

Размер первого элемента равен 130 * 130 px

Размер второго элемента считается по-другому. Из-за значения box-sizing: border-box браузер воспринимает width и height как конечные размеры. То есть в них уже включены размеры padding и border

Таким образом размер второго элемента 100 px

Что с контентной частью?

Во втором случае размер блока сохраняется равным 100px за счет сужения контентной части. То есть 100px - border(5px*2) - paddint(10px*2) = 70px

Что делать? Чтобы было меньше ошибок использовать box-sizing: border-box в обнуляющем файле css.

Виды элементов и их свойства

Всего есть 3 вида элементов

  • Блочный (block) - занимает всю ширину родителя и по умолчанию равна 100%. По умолчанию height блока равна нулю и равна высоте+ контента.
  • Строчно-блочный (inline-block) - height и width равна высоте и ширине контента. Можно принудительно задать ширину и высоту блока. По умолчанию height также равна 0.
  • Строчный (inline) - height и width равна высоте и ширине контента. Нельзя задать принудительно ширину и высоту. Height по умолчанию равна 0.

Свойства блоков

  • border - Устанавливает границу элемента. Записыватеся как border:толщина стиль(solid, dotted, dushed, double) цвет;
  • border-radius - Радиус скругления уголков блока. Записывается как border-radius: верхний-левый верхний-правый нижний-правый нижний-левый;
  • outline - Универсальное свойство, одновременно устанавливающее цвет, стиль и толщину внешней границы на всех четырех сторонах элемента. Записывается outline: толщина стиль(solid, dotted, dushed, double) цвет;
  • box-shadow - Добавляет тень элементу. Записывается как box-shadow: сдвиг_по_гориз сдвиг_по_верт радиус(размытие) расстояние(размер) цвет;
  • opacity - Определяет уровень прозрачности элемента. Значение может быть равно от 0 до 1.
  • visibility - Похоже на opacity. Но имеет всего 2 значения visible (по умолчанию) и hidden.
